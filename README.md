## Description

Little but complete CRUD API built with <a href="http://nestjs.com/" target="_blank">NestJS</a>

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

```

## Run tests

```bash

# e2e tests
$ pnpm run test:e2e

```
